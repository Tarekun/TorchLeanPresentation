This is a quick demo project I wrote for a presentation given on fully typed checked neural networks using [TorchLean](https://lean-dojo.github.io/TorchLean/).

The project contains the implementation of a small MLP with one single hidden layer of 30 neurons trying to predict a $f:\mathbb{R}^{10}\to\mathbb{R}$ defined as $x\mapsto(\sum_i x_i)^2$. Data for training is generated by sampling a random batch points from $\mathbb{R}^{10}$ and Gaussian noise is applied to the value.

The `pytorch` directory contains an implementation of this neural network using the de-facto standard library PyTorch.

More interestingly, the `lean-mlp` directory contains the implementation of the same network and experiment using [TorchLean](https://lean-dojo.github.io/TorchLean/), a library that adds support for torch in the theorem proving language [Lean](https://lean-lang.org/).

The aim of the presentation is to show that we can write code with much stronger guarantees than what python gives us, structurally disabling a whole class of annoying bugs.
